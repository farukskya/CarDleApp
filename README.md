# 🚗 CARDLE - Premium Araba Tahmin Oyunu

Dünyaca ünlü Wordle oyununun Türkiye sokaklarına uyarlanmış, Flask (Python) ve JavaScript ile geliştirilmiş premium bir versiyonudur. Oyunda Fiat Linea 1.3 MultiJet'ten Chevrolet Cruze'a kadar 200 farklı popüler araç verisi yer almaktadır.

---

## 🚀 Özellikler

* **Geniş Araç Havuzu:** Türkiye'de sıkça görebileceğiniz 200 popüler araç modeli.
* **Akıllı Öneri Listesi:** Arama kutusuna yazmaya başladığınızda araçları otomatik olarak listeler ve seçmenizi kolaylaştırır.
* **Anlık Geri Bildirim:** Marka, Model, Kasa, Yakıt, Motor, Yıl ve Ülke kriterlerine göre yeşil (doğru) ve kırmızı (yanlış) renkli analiz sunar.
* **Arkadaşlarla Paylaşım Desteği:** Ngrok entegrasyonu sayesinde yerel ağdan internete açılarak arkadaşlarla WhatsApp üzerinden paylaşılıp oynanabilir.

---
